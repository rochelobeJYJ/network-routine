# Release Notes

## v1.1.2

수동 전환 오류를 수정한 버전입니다.

### 변경 사항

- DNS 변경 명령을 Windows에서 지원되는 `netsh interface ipv4 set/add dnsservers` 형식으로 수정
- DNS 서버 설정 시 검증 지연으로 실패하지 않도록 `validate=no` 적용
- 수동 전환 시 작업 스케줄러 재등록을 수행하지 않도록 분리
- `알 수 없는 오류` 대신 실패한 명령과 종료 코드를 표시하도록 개선

### 배포 파일

- `NetworkRoutine.exe`

## v1.1.0

자동 전환 방식을 재구성한 배포 버전입니다.

### 변경 사항

- `분 단위 감시`를 제거하고 출근/퇴근 경계 시각에만 자동 전환하도록 변경
- 로그인, 잠금 해제, 콘솔 복귀 시 현재 시간대 기준으로 한 번만 자동 보정
- 같은 시간대에서는 자동 전환을 한 번만 처리하도록 변경
- 사용자가 수동 전환한 현재 시간대에는 자동이 다시 덮어쓰지 않도록 변경
- 작업 상태 UI를 `정시/로그온/복귀` 기준으로 정리
- GitHub 저장소에서 배포에 불필요한 `.bat` 파일 제거

### 배포 파일

- `NetworkRoutine.exe`

## v1.0.0

첫 공개 배포 버전입니다.

### 포함 기능

- 내부망/외부망 수동 전환
- 내부망 설정값 GUI 입력 및 로컬 저장
- 요일별 출근/퇴근 시간 기반 자동 전환
- 로그인 직후 자동 상태 보정
- 작업 등록 상태 표시
- 배터리 사용 중에도 계속 동작하는 자동 루틴
- Python 없이 실행 가능한 단일 exe 배포

### 배포 파일

- `NetworkRoutine.exe`

### 참고

- 사용자 배포본은 로그 파일을 기본 생성하지 않습니다.
- 설정 파일은 첫 실행 후 exe와 같은 폴더에 자동 생성됩니다.
